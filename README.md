# CeZIS Team submission for the The George B. Moody PhysioNet Challenge 2026

## Abstract
Within The George B. Moody PhysioNet Challenge 2026, we developed a random forest model classifying Cognitive Impairment from demographic metadata (age, gender, BMI). The prediction model is random forest model with 1000 decision trees predicting the label.
